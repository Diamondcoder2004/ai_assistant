# Узнать статус заявки

Сервис проверки статуса заявки на технологическое присоединение.

Для проверки статуса необходимо ввести индивидуальный номер заявки, присвоенный при регистрации.

Статус заявки также можно узнать через Личный кабинет потребителя услуг по технологическому присоединению.
