# Калькулятор необходимой мощности

С помощью калькулятора можно оценить потребность в электрической мощности и использовать расчет при подаче заявки на технологическое присоединение.

## Как пользоваться

Для расчета нагрузки необходимо указать количество объектов, затем заполнить суммарную нагрузку одного объекта (в кВт).

Суммарная нагрузка может быть определена как общий объем потребляемой мощности всех электроприборов одного объекта.

## Результат расчета

Итоговая нагрузка выдается в двух значениях:
- На шинах 0,4 кВ с учетом совмещения, кВт
- На шинах 10 кВ с учетом совмещения, кВт
