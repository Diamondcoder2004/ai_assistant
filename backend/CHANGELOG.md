## [0.1.0] - 2026-03-23

### Добавлено
- ✅ Первый релиз Agentic RAG
- ✅ QueryGeneratorAgent — генерация поисковых запросов
- ✅ SearchAgent — агент поиска с Tool Calling
- ✅ ResponseAgent — генерация ответов
- ✅ SearchTool — гибридный поиск (4 компонента)
- ✅ Системный промпт для дружелюбного общения
- ✅ Промпт для генерации запросов
- ✅ Benchmark для оценки качества
- ✅ Юнит тесты для всех компонентов
- ✅ Логирование работы
- ✅ Документация (README, QUICKSTART, ARCHITECTURE)

### Изменения
- ✅ Обновлены API endpoints bashkir_rag для передачи content и category
- ✅ Обновлена схема SourceResponse

### Технические детали
- Поддержка int8 эмбеддингов Perplexity
- Гибридный поиск: pref + hype + lexical + contextual
- Автоматический подбор весов через LLM
- Генерация 2-3 запросов из одного вопроса
- Стратегии поиска: concat / separate
- Уточняющие вопросы при неопределённости
