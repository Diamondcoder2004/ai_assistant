# Горячие клавиши

## Настройки по умолчанию

| Действие | Комбинация | Описание |
|----------|------------|----------|
| **Отправить сообщение** | `Enter` | Отправка вопроса в чат |
| **Новая строка** | `Shift+Enter` | Перенос строки в поле ввода |
| **Новый чат** | `Ctrl+N` | Начало нового диалога |
| **История** | `Ctrl+H` | Показать/скрыть панель истории |
| **Фокус на ввод** | `Ctrl+L` | Переместить фокус в поле ввода |
| **Источники** | `Ctrl+S` | Показать/скрыть панель источников |
| **Копировать ответ** | `Ctrl+Shift+C` | Копировать последний ответ в буфер |

## Как изменить горячие клавиши

1. Откройте **Настройки** (кнопка ⚙️ в шапке)
2. Перейдите на вкладку **⌨️ Горячие клавиши**
3. Кликните на поле с текущей комбинацией
4. Нажмите новую комбинацию клавиш
5. Если комбинация допустима, она сохранится автоматически

## Допустимые комбинации

### ✅ Можно использовать

- **Модификаторы:** `Ctrl`, `Alt`, `Shift`, `Win`
- **Буквы:** `A-Z`
- **Цифры:** `0-9`
- **Функциональные:** `F1-F12`
- **Специальные:** `Enter`, `Escape`, `Tab`, `Стрелки`, `Home`, `End`, `PageUp`, `PageDown`, `Delete`, `Backspace`

**Примеры допустимых комбинаций:**
- `Ctrl+K`
- `Alt+Shift+S`
- `Ctrl+F5`
- `Ctrl+Shift+H`

### ❌ Нельзя использовать

- Только `Enter` или `Пробел` с модификаторами (конфликт с браузером)
- Одинаковые комбинации для разных действий
- Недопустимые клавиши

## Сброс настроек

Чтобы сбросить все горячие клавиши к настройкам по умолчанию:

1. Откройте **Настройки** → **Горячие клавиши**
2. Нажмите кнопку **🔄 Сбросить**
3. Все комбинации вернутся к значениям по умолчанию

## Конфликты

Если вы назначили одну и ту же комбинацию на два действия:

- Система покажет предупреждение **⚠️ Конфликт с: [действие]**
- Новая комбинация не сохранится пока вы не измените её
- Чтобы исправить, выберите другую комбинацию или измените конфликтующую

## Валидация

При вводе комбинации система автоматически проверяет:

1. ✅ Есть ли хотя бы одна клавиша
2. ✅ Допустимые ли модификаторы используются
3. ✅ Нет ли конфликта с другими горячими клавишами
4. ✅ Не конфликтует ли с системными комбинациями браузера

Если проверка не пройдена, поле подсветится **красным** с сообщением об ошибке.

## Советы

### 💡 Оптимальные комбинации

Используйте мнемонические комбинации (по первым буквам):

- `Ctrl+N` — **N**ew chat (новый чат)
- `Ctrl+H` — **H**istory (история)
- `Ctrl+S` — **S**ources (источники)
- `Ctrl+L` — **L**ocate (фокус на ввод)

### 💡 Избегайте конфликтов

Не используйте комбинации которые уже заняты браузером:

- `Ctrl+T` — новая вкладка
- `Ctrl+W` — закрыть вкладку
- `Ctrl+R` — обновить страницу
- `Ctrl+P` — печать

### 💡 Используйте модификаторы

Для часто используемых действий используйте простые комбинации:
- `Enter` — отправить (самое частое)

Для редко используемых — с модификаторами:
- `Ctrl+Shift+C` — копировать ответ (реже)

## Сохранение настроек

Настройки горячих клавиш сохраняются **автоматически** при изменении:

### Для авторизованных пользователей:
- ✅ Сохраняются в **Supabase** (база данных)
- ✅ Синхронизируются между устройствами
- ✅ Привязаны к аккаунту пользователя
- ✅ Кэшируются в **localStorage** для скорости

### Для неавторизованных:
- ✅ Сохраняются только в **localStorage** браузера
- ❌ Не синхронизируются между устройствами
- ❌ Сбрасываются при очистке кэша браузера

**Рекомендация:** Войдите в систему чтобы настройки сохранялись в облаке!
